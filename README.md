# ACT Fitness & Diary App

แอปพลิเคชันสำหรับช่วยจัดการการใช้ชีวิตในปัจจุบัน  
รวมการออกกำลังกาย (Workout) + บันทึกแผนชีวิต (Diary) + แนะนำสุขภาพจากค่า BMI  

---

## 📱 Features

- 🏠 Home
  - คำนวณ BMI
  - แนะนำอาหารในแต่ละมื้อ
  - แนะนำตามข้อมูลผู้ใช้

- 📅 Diary
  - เพิ่มสิ่งที่ต้องทำล่วงหน้า 7 วัน
  - วางแผนการออกกำลังกาย

- 🏋️ Workout
  - Running Mode (จับเวลา + คำนวณ Pace)
  - Weight Training (แนะนำท่า + บันทึก Set / Rep)

---

## 🛠 Tech Stack

- Android (Java / Kotlin)
- Firebase
  - Authentication
  - Firestore Database

---

---
###ขันตอนสำคัญในการตั้ง SHA-1
1.ใน Android Studio เครื่องใหม่ ให้กดแถบ Gradle (อยู่ขวาสุดของหน้าจอ)
2.ไปที่ app > Tasks > android > signingReport (ดับเบิลคลิก)
3.ดูที่ช่อง SHA-1 ในหน้าต่างด้านล่าง แล้วคัดลอกรหัสนั้นไว้
4.ไปที่ Firebase Console เข้าไปที่โปรเจกต์ของคุณ
5.ไปที่ Project Settings (ไอคอนฟันเฟือง) > แถบ General
6.เลื่อนลงมาที่แอป Android ของคุณ กดปุ่ม "Add fingerprint" แล้ววางรหัส SHA-1 ของเครื่องใหม่ลงไป
